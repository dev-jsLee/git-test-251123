# git 교육
## git 기초 명령어
1. 깃 환경을 세팅하려면 초기 준비 상태를 만들어야 한다.
- `git init` : 해당 폴더를 git으로 관리하겠다-를 선언
2. 깃을 쓰는 사용자를 설정해주어야 한다.
- `git config`: git과 관련된 설정을 하는 명령어다.
	- `git config --local user.name "jslee7518"`: 해당 폴더에서만(--local) 사용자 이름을(user.name) 설정하는데, 그 값은 "jslee7518"로 설정한다
	- 글로벌 세팅: 해당 PC를 혼자 사용하는 경우, 모든 깃으로 관리하는 폴더마다 사용자 설정하기가 귀찮다.
	- `git config --global user.name "jslee7518"`: 해당 PC 어디서든지(--global) 모두 같은 설정을 사용한다.
- user.name "닉네임 입력"
- user.email "깃허브에 등록한 이메일 작성"
	- ※주의사항: 큰따옴포""를 반드시 써야 한다.

1. 깃 저장소를 선언한다(`git init`)
2. 사용자 정보를 설정한다(`git config user.name / user.email`)
3. 변경사항이 있는 파일을 추가(=> staging)
4. 스테이징된 변경사항을 확정한다(=>commit)
5. 커밋을 하여 버전을 확정지을 때 해당 버전의 변경사항에 대해 메모를 남길 수 있다.(`git commit -m "메시지"`)
